〜体験会に参加してどうだったか〜
体験会に参加して、画面を共有しながら、一緒に作品を作るのが純粋に楽しかったです。このwebサイトは最初はデフォルトで用意されていました。そこから、デザインを変えたり、文章をかえたりしました。お互いはじめましての状態で、学年も違いますが、良い経験になりました。

〜体験会でがんばったこと〜
積極的に発言したことです。例えば、自分で作った作品（敬語English）を見せたり、自己紹介するときも一番に発言したりしました。体験会の中で僕が一番年下でしたが、気にせずに積極的に発言できたのも、周囲の環境がよく、発言を歓迎してくれる状態だったので、発言できました。もし、緊張感のある中だったら、おそらくできてないと思います。なので、体験会に参加して、交流して、作品を作っていくことは良い経験になりました。


〜webサイトをデフォルトから修正していく中で工夫した点〜
名前が見えやすいように下線を引いたり、border-radiusの数値が大きかったので、少なくして、ちょっとだけ丸くした感じにたりしました。理由は、まるいのもいいのですが、僕的にはまるすぎだと思って、修正しました。さらに、Twitterや敬語Englishのボタンもちゃんとリンクに飛べるようにしました。
